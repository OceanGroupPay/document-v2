**MyPay88**允许开发者通过API来进行交易，主要关注通过RESTful API或SDK进行支付和提款。 所有MyPay88产品都有其大部分核心功能的 API。希望能通过MyPay88的全面API来重新构想您的业务。

## 环境

目前提供了两个环境来测试功能并确保无缝完成集成，有**沙盒**环境和**生产**环境。

### 沙盒 环境

登录系统 -> https://demo.mypay88.asia/ <br />
API 接口服务器 -> https://testapi.mypay88.asia <br />

### 生产 环境

登录系统 -> https://merchant.mypay88.asia
API 接口服务器 -> https://api.mypay88.asia

**开始之前**

前往[商户平台](https://merchant.mypay88.asia) 注册成为商家

提交商家门户中显示的所有必需文件，并获得我们管理员的批准。 如果您需要更多帮助，请通过 <helpcare@mypay88.co> 联系我们。
