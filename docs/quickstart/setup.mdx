### Step 1. Login Merchant Portal
Access <EnvVariable domainPrefix={`https://merchant.`} envKey={"ENV_DOMAIN"} />, then on the **Sidebar** from the left, go to **Development Manage** -> **Development Settings**
<Image envKey={"ENV_DOMAIN"} />

### Step 2. Generate Credentials
If this is Merchant first visit to this page, please fill up the *Redirect URL, Callback URL, Withdraw Callback URL*, and generate for **Key, Secret**, along a keypair of **PublicKey** and **PrivateKey**.

:::note
Please Keep the **Private Key** safe as <EnvVariable envKey={"ENV_BRAND"} /> will not be storing the Private Key.
:::
